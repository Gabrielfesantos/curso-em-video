Atribuindo Cores com CSS

Podemos atribuir cores usando CSS de formas diferentes, porém com o mesmo resultado e ,dependo do metodo empregado, com mais versarilidade.Os métodos são: Nome da Cor, RGB(), HSL(0,%,%) e Código Hexadecimal.

Nome da Cor

é a forma mais simples de atribuir uma cor, apenas devemos digitar digitar o nome da cor desejada em inglês, como blue, red, green, yellow, etc.

RGB(0, 0, 0)

Podemos atribuir as cores usando rgb(0, 0, 0). Devemos dizer dentro dos parenteses a quantidade de cada cor queremos, de 0 à 255.

HSL(0, %, %)

Usando hsl(0, %, %) devemos informar a tonalidade desejada, a porcentagem da saturação e da luminosidade.

Código Hexadecimal

Podemos dizer depois da # o quanto de cada cor nós precisamos para formar a cor desejada.
